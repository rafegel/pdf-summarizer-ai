# 📄 PDF Summarizer AI

Automatically summarize any PDF file using Python and Groq LLM API.

## 🚀 What it does
- Reads any PDF file
- Sends the content to Groq AI
- Returns a clean, concise summary

## 🛠️ Technologies
- Python
- Groq API (LLaMA 3.1)
- PyMuPDF

## ⚙️ Setup

1. Clone the repo
2. Install dependencies:
   pip install groq pymupdf python-dotenv
3. Create a `.env` file:
   GROQ_API_KEY=your_key_here
4. Run:
   python main.py

## 💡 Use case
Great for summarizing research papers, reports, or any long PDF document in seconds.
